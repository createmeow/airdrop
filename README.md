# AirDrop

基于 NeoForge 1.21.1 的空投模组。

## 功能

- **信号枪**：使用信号枪发射信号弹，召唤空投箱降落。
- **高级信号枪**：召唤更高品质的空投。
- **三种品质空投**：普通、中级、高级，战利品数量和稀有度随品质提升。

## 配置

配置文件位于 `config/airdrop/airdrop-common.toml`，可调整：

- 空投箱存在时间
- 基地最小距离等

战利品表位于 `config/airdrop/loot.json`，首次启动时会自动生成默认值。

## 联动模组

本模组对以下模组提供可选联动支持（均为可选依赖，未安装时自动跳过）：

| 模组 | 普通 | 中级 | 高级 |
|------|------|------|------|
| **BaseCore** | 铜丝、电缆、芯片、天线、小型/中型零件包、零件 | 精制芯片、电子元件、模块基底、中型零件包、零件 | 各类模块、伪装装置、大型零件包、零件 |
| **RealityValue** | 草药、破布、绷带 | 朗姆酒、抗生素、急救针 | - |
| **ColdSweat** | 山羊皮、疣猪兽皮、变色龙皮 | 温度计、水袋、灵魂灯 | - |
| **ThirstWasTaken** | 陶碗、陶土碗、陶土水碗 | - | - |
| **NightVisionDevice** | - | - | 夜视仪 |
| **Soulbound** | - | - | 灵魂绑定附魔书 |
| **cmmsg** | 通讯器 | 通讯器 | 通讯器 |
| **Create** | 安山合金、铜板、铁板 | 黄铜锭、玫瑰石英、传动杆、齿轮、大齿轮、扳手 | 黄铜/安山/铜机壳、精密构件、电子管、黄铜板 |

